# EmptyState

A centered icon/title/body placeholder for a list or table with nothing to show, with an
optional call-to-action.

## User story

**US-CW-025 — Component library — organisms & decorators**

As a clearline-web engineer, I want DataTable/ApprovalActionBar/BulkActionResult/EmptyState and
the AppShell decorators available as typed React components, so that I
can assemble a full screen (selection, sorting, approval flows, partial-failure summaries) by
composing already-built pieces instead of writing table selection/sort logic or approval-limit
disabling logic from scratch in each feature epic. EmptyState has no acceptance criterion of its
own in this story; it is governed by the tier's shared constraints below.

## Acceptance criteria (shared across this tier)

- AC-01 — AC-02 — DataTable's select-all toggle and vendor-column sort.
- AC-03 — ApprovalActionBar disables Approve with a stated reason when `canApprove` is unset.
- AC-04 — BulkActionResult excludes failures from the committed count and lists each reason.
- AC-05 — AppShell owns theme state and requires zero additional per-page token setup.
