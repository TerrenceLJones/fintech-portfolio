# AlertModal

A general-purpose dialog built on Radix `Dialog`, with `accent`/`negative`/`warning` tones for
its icon/title treatment.

## User story

**US-CW-024 — Component library — molecules**

As a clearline-web engineer, I want AlertModal/ConfirmationDialog and the other 6 Tier-2 molecules
available as typed React components built on Radix Dialog, so that every irreversible-action
confirmation in the app gets the same focus-trapped, Escape-cancels, countdown-gated behavior for
free, instead of each feature epic re-implementing dialog accessibility by hand.

## Acceptance criteria

- **AC-01 — AlertModal.** Given an AlertModal with `open={true}`, when it renders, then it exposes
  `role="dialog"`, traps focus, and closes via Escape or the Cancel button — all inherited from
  Radix Dialog, not hand-rolled.
- AC-02 — ConfirmationDialog's `countdown` gates the confirm button and re-arms on every open.
- AC-03 — VirtualCard hides the derived remaining-limit figure entirely when frozen.
- AC-04 — Timeline shows before → after diffs with an arrow between them.
- AC-05 — AIInsightCard's primary/secondary actions independently invoke their own callbacks.

### Edge case covered here

AlertModal renders nothing at all when `open={false}` — no hidden-but-present DOM.
