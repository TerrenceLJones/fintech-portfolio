# Timeline

An audit-trail list of actor/action/time entries, with an optional before → after diff per
entry.

## User story

**US-CW-024 — Component library — molecules**

As a clearline-web engineer, I want Modal/ConfirmationDialog and the other 6 Tier-2 molecules
available as typed React components built on Radix Dialog, so that every irreversible-action
confirmation in the app gets the same focus-trapped, Escape-cancels, countdown-gated behavior for
free, instead of each feature epic re-implementing dialog accessibility by hand.

## Acceptance criteria

- AC-01 — Modal exposes `role="dialog"`, traps focus, and closes via Escape/Cancel.
- AC-02 — ConfirmationDialog's `countdown` gates the confirm button and re-arms on every open.
- AC-03 — VirtualCard hides the derived remaining-limit figure entirely when frozen.
- **AC-04 — Timeline.** Given a Timeline with entries that include `diffFrom`/`diffTo`, when
  rendered, then the before → after values are shown with an arrow between them, distinct from
  entries with no diff.
- AC-05 — AIInsightCard's primary/secondary actions independently invoke their own callbacks.

### Edge case covered here

A single Timeline mixing diffed and non-diffed entries is exercised as one story, so the two
entry shapes must render correctly side by side.
